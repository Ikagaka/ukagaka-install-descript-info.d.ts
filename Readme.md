Installation
--------------------------

```
npm install ukagaka-install-descript-info
```

Usage
--------------------------

```javascript
import {UkagakaInstallInfo} from "ukagaka-install-descript-info";

const ukagakaInstallInfo = UkagakaInstallInfo.parse(data);
```
